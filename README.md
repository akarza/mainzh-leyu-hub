# mainzh-leyu-hub

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放各类独立的 HTML 页面文件。这些页面可能涉及不同的主题或用途，彼此之间没有强关联，仅作为页面内容归档与展示的集合地。

每个 HTML 页面均独立存在，不依赖特定后端服务，可直接在浏览器中打开或通过静态托管服务访问。

## 目录说明

```
.
├── pages/          # 存放所有独立的 HTML 页面文件
├── assets/         # 页面所需的共用资源（如样式、脚本、图片等）
├── index.html      # 仓库首页或导航页面（如有）
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件对应一个独立页面，文件名通常反映页面主题。
- **assets/**：存放页面可能引用的共用资源文件，如 CSS、JavaScript、字体或图片。
- **index.html**（可选）：如果存在，通常用于提供页面列表或导航功能。

## 页面归档说明

本仓库中的 HTML 页面主要用于归档和展示。每个页面在被提交前通常经过基本测试，确保可正常渲染。

- 页面之间相互独立，不共享状态或数据。
- 部分页面可能引用外部资源（如 CDN 链接），但尽量保证离线可查看。
- 页面内容可能随时间更新或替换，历史版本可通过 Git 提交记录追溯。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 `.html` 文件即可查看。
3. 也可将整个仓库部署到任意静态托管服务，实现线上访问。
